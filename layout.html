Selamat Datang di Tampilan Layout

Bagian ini adalah Main Content, yang merupakan tempat menaruh isi konten utama pada website kita. Di bagian ini biasanya pengunjung akan menghabiskan waktu mereka untuk membaca maupun melihat detail spesifik dari halaman bersangkutan.

Menurut instruksi, kita membuat struktur dasar website menggunakan tag <table> HTML. Komponen layout yang dirancang terdiri dari:

  1. Header: Di bagian atas sebagai penanda brand atau judul.
  2. Sidebar: Berada di kiri, berguna sebagai area navigasi atau fitur terpisah.
  3. Main Content: Area paling luas yang memuat konten spesifik per halamannya.
  4. Footer: Bagian bawah halaman untuk keterangan lisensi atau tautan penutup.
